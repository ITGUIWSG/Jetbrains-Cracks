# Jailbreaking-Jetbrains
## Jetbrains全家桶破解

感谢制作补丁的大佬知了    
使用方法很简单，把“jetbrains-agent-latest.zip”拖入就行，具体看[博客内容](https://zhile.io/2018/08/25/jetbrains-license-server-crack.html)和readme
